Academic Performance Analyzer

A terminal-based Python utility designed to collect student scores, evaluate grading scales, and generate a class performance summary report. 

 Features
* **Dynamic Collection:** Captures custom inputs for student names and numerical scores.
* **Automated Grading:** Utilizes conditional logic to assign standard letter grades (A, B, C, D, F).
* **Statistical Insights:** Dynamically calculates and formats the overall class average.
* **Clean Data Modeling:** Manages records using lists and dictionaries (structurally similar to JSON arrays).
